# Let's Play Rock Paper Scissors!

Rock Paper Scissors is a timeless classic, and I really enjoyed writing this fun little program to play it in the browser console.

## How it Works

The program essentially prompts the user to choose either "Rock", "Paper", or "Scissors." It then compares the user's input to the Computer's choice, which is randomly selected at each round. After five rounds, the number of user wins is compared to the computer's wins, and an ultimate winner is proclaimed.

## Technology

The ultimate goal of this project was to implenent various **JavaScript** concepets, especially **functions** in order to create this console game.
